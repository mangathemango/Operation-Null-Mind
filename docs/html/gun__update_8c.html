[PostUpdate] Updates the gun's state, position, and particles

Updates gun state.

Updates the gun's position relative to the player, calculates the angle based on mouse position, and updates particle emitter positions for muzzle flash and casings. Flips the gun sprite horizontally when aiming to the left.
