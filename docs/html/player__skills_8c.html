[Utility] Activates the Crashout skill, sacrificing health for damage

Activates crashout ability.

Takes 25% of the player's current health and gives a 2x damage multiplier for 15 seconds. Has a cooldown before it can be used again.

Returns
int Status code (0 for success, 1 for on cooldown)
