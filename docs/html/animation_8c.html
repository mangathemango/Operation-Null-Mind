Detailed Description

Sprite animation system implementation.

Implements a flexible animation system that supports spritesheet-based animations with multiple clips, variable frame rates, and playback controls.

Adds an animation clip from a spritesheet grid.

Adds a new animation clip from a grid-based spritesheet.

Creates a new clip by extracting frames from the spritesheet using a grid-based layout. Frame indices start from 0, reading from left to right, then top to bottom.

Parameters
animationThe animation to add the clip to
nameName of the clip (e.g., "idle", "walk")
startFrameIndexIndex of the first frame in the clip
endFrameIndexIndex of the last frame in the clip
frameDurationTime between frames in seconds
loopingWhether the animation should loop
Returns
int 0 on success, 1 on failure
