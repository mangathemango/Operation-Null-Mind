void Gun_UpdatePosition ( GunData *  gun,
Vec2  ownerPosition,

Vec2  gunOffset 
)

Updates gun position, orientation and associated particle emitters.

Parameters
gunPointer to the gun data structure
ownerPositionPosition of the entity that owns the gun
targetPositionPosition to aim at (usually player position)
gunOffsetAdditional offset from owner position

A generalized function to handle gun positioning, orientation, and particle emitter updates for any entity (player or enemy) that uses a gun. This eliminates duplicate code across different enemy types.
