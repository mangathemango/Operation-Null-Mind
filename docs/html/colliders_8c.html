Collision detection system.

Provides functionality for hitbox-based collision detection between game objects with layer-based filtering.

[PostUpdate] Checks if a collider is intersecting with any of its collider layers.

Check for collisions with a collider.

This function checks for collision between a collider and everything else in the ColliderList array. Any colliders whose layer is not in the input collider's collidesWith section will be ignored.

Parameters
colliderThe input collider
checkResultThe checkResult of the collider, which includes 2 members: checkResult.objects: an array of detected colliders checkResult.count: number of collisions detected
Returns
true if collision detected, false otherwise
