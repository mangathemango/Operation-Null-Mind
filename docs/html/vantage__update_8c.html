void Vantage_UpdateLazer ( EnemyData *  data)

Updates the laser targeting system for the Vantage enemy.

Calculates the laser beam's path from the gun to the first obstacle or player, handling collision detection and damage application. The laser is used for both visualization (showing where the Vantage is aiming) and for dealing damage when in shooting state.
