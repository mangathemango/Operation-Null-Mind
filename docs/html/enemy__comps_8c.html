EnemyData * Enemy_SelectRandomEnemyInComp ( EnemyComp *  comp)

Selects a random enemy type from the given enemy composition.

This function randomly selects an enemy type from the provided enemy composition, taking into account the maximum spawn count for each enemy type. It returns a pointer to the enemy data for the selected enemy type.

Parameters
